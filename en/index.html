    MCP-ECharts

    MCP server for chart generation based on ECharts

    Quick StartDocumentation

    Easy to Use

    • Supports npx for one-click local usage

    • Supports quick deployment of HTTP version for multi-user needs

    Variety of Chart Types

    Supports various common chart types to meet diverse data visualization needs

    Bar Chart

    Supports basic bar charts, stacked bar charts, and more styles

    Line Chart

    Supports smooth line charts, area charts, and various configurations

    Pie Chart

    Supports pie charts, donut charts, rose charts, and more styles

    Scatter Chart

    Supports single-axis, Cartesian, geographic scatter charts, and more coordinate systems

    Calendar Chart

    Supports calendar scatter charts to visualize time series data

    Map Customization

    Supports custom maps with dynamic GeoJSON registration

    Theme Support

    Supports dark/light theme switching

    Key Features

    Parameter validation, image hosting, server-side rendering

    • Parameter Validation

      Automatic JSON Schema generation based on Zod validation

    • Image Hosting

      Built-in basic static hosting service, ready to use

    • SSR

      Server-Side Rendering for high-performance chart generation
